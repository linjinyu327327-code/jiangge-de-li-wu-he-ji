# 姜哥的礼物合集

这里是姜哥的两个个人作品集页面。

## 作品列表

### 1. 给世界的礼物-泰国 | 周期与自由
- 目录: `给世界的礼物-泰国/`
- 文件: `index.html`
- 主题: 周期与自由
- 描述: 一篇关于人生周期与追求自由的深度思考文章，采用深色调设计，优雅的排版和动画效果。

### 2. 命与苦海 | 姜哥认知备忘录
- 目录: `命与苦海/`
- 文件: `index.html`
- 主题: 命与苦海 - 认知备忘录
- 描述: 姜哥的个人认知思考记录，探讨生命与苦难的深刻话题，设计风格沉稳内敛。
